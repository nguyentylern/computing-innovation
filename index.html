napster

description

Napster was a peer to peer file sharing network designed around sharing music files. A peer to peer (P2P) file sharing network is a system of connecting computers that allows one user to access files on other computers on the network. In P2P file sharing, there is no one host of files that people can access; people access files from people also on the network. In other words, there is no middle man: Napster was the means of connecting them.

purpose

Napster's purpose was to provide an accessible means to secure music.
